# Amazon Sales Analysis

This project analyzes Amazon sales data using SQL and Power BI to identify sales trends, revenue distribution, and order patterns.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel

## Project Insights
- Analyzed monthly sales trends
- Identified top performing cities by revenue
- Evaluated sales distribution by product category
- Analyzed order status distribution
